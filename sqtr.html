Related Useful Tools

Where you will find related tools?

These are some useful related tools that can ease your review tasks:

  • The Polyglot: Part of an open systematic review accelerator tools including a query manager. The Polyglot helps you translate your queries to different databases.
    Availability: Free online
    Tool Type: Query translation
    Features: Multi-database support, user-friendly interface.
  • Systematic Review (SR) Toolbox: A comprehensive collection of applications for conducting systematic reviews.
    Availability: Free online
    Tool Type: Review management
    Features: Various systematic review tools, user resources.
  • App2DSearch: A graphical search query builder tool designed to streamline your search process.
    Availability: Free online
    Tool Type: Query builder
    Features: Visual query construction, intuitive interface.
  • PubMed PubReMiner: A text mining tool that analyzes PubMed search results for word frequency and relevant MeSH headings.
    Availability: Free online
    Data Source: PubMed
    Tool Type: Text frequency analysis
    Features: Frequency tables, direct connection to PubMed.
    URL: http://hgserver2.amc.nl/cgi-bin/miner/miner2.cgi
  • PubVenn: A visualization tool that generates Venn diagrams for exploring PubMed citations based on multi-term searches.
    Availability: Free online
    Data Source: PubMed
    Tool Type: Visualization
    Features: Interactive Venn diagrams, citation set exploration.
    URL: https://pubvenn.appspot.com/
  • Coremine Medical: A text-mining tool providing a graphical representation of relationships in health and medical literature.
    Availability: Register for a free account
    Data Source: PubMed
    Tool Type: Visualization, clustering
    Features: File upload, search history, alerts.
    Instructions: Help
  • Carrot2: An open-source results clustering engine that organizes search results into topics.
    Availability: Free online, downloadable
    Data Source: PubMed
    Tool Type: Clustering
    Features: Boolean searching, simple interface.
  • HelioBLAST: A text similarity engine that finds articles similar to the submitted query within Medline/PubMed.
    Availability: Free online
    Data Source: PubMed
    Tool Type: Word analysis
    Features: Similarity analysis, links to article records.
  • Yale MeSH Analyser: A tool for comparing article indexing based on MeSH vocabulary.
    Availability: Free online
    Data Source: PubMed
    Tool Type: Comparison
    Features: Easy-to-view comparisons, export options to Excel or HTML.
  • MeSH on Demand: Analyzes submitted text to identify MeSH vocabulary and suggests relevant citations.
    Availability: Free online
    Data Source: PubMed
    Tool Type: Text analyzer
    Features: Highlights terms in text, links to similar citations.
Some Useful Databases

Some Useful Databases:

These are some useful databases that utilize free text search:

  • Semantic Scholar: A free, AI-powered research tool that helps scholars find relevant papers and understand the context of research. It uses machine learning to extract key concepts and relationships from the literature.
    Availability: Free online
    Data Source: Various academic publications
    Type: Research database
    Features: Citation graph, paper recommendations, and search by key concepts.
  • Europe PMC: A free database of life sciences and biomedical literature. It provides access to millions of articles, including full-text content and associated datasets.
    Availability: Free online
    Data Source: PubMed Central, Europe PMC
    Type: Literature database
    Features: Advanced search options, citation tracking, and access to related datasets.
  • ResearchRabbit: A tool that helps researchers find relevant literature based on their existing papers or concepts. It visualizes connections and suggests additional readings.
    Availability: Free online
    Data Source: Academic publications
    Type: Literature discovery tool
    Features: Visual mapping of research topics and personalized literature recommendations.
  • Connected Papers: A unique, innovative tool for researchers to create a visual graph of related papers, helping users understand the broader context of their research.
    Availability: Free online
    Data Source: Various academic sources
    Type: Literature visualization tool
    Features: Graph-based exploration, easy navigation through related works.
  • ResearchGate: A professional network for researchers to share papers, ask and answer questions, and find collaborators. It hosts a vast repository of research outputs.
    Availability: Free online (registration required)
    Data Source: User-generated content and publications
    Type: Academic social networking site
    Features: Full-text access to articles, Q&A platform, and collaboration opportunities.
  • Health Systems Evidence: A comprehensive database that provides access to systematic reviews and other evidence related to health systems and policy.
    Availability: Free online
    Data Source: Systematic reviews and health policy literature
    Type: Evidence database
    Features: Searchable database, filtering by intervention type, and evidence summaries.
